# CycleGo

## Setup Client
<ol>
<li>cd client</li>
<li>npm install</li>
<li>cd npm start</li>
</ol>

## Setup Server 
<ol>
<li>cd server</li>
<li>bundle install</li>
<li>bundle exec rake server</li>
</ol>

## Summary
After both servers are running you will have the full features of the application.
